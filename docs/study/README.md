# Study Notes

Folder này chứa tài liệu học tập cho các khái niệm Spring Boot và clean architecture được dùng trong project.

Nguyên tắc:

- Mỗi khi thêm một loại class/layer quan trọng mới, kiểm tra folder này trước.
- Nếu đã có tài liệu phù hợp thì không tạo file trùng.
- Nếu chưa có, tạo study note ngắn gọn nhưng đủ đọc lại để học.
- Study note nên bám vào code thật trong project, không viết lý thuyết quá xa.

Các tài liệu hiện có:

- `JPA_ENUM_AND_ENTITY_STUDY.md`: giải thích JPA enum và entity qua các enum domain và `Wallet` entity.
